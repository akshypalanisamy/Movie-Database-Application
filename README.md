# RESTful Application
A web application that displays films and its category in a database, allowing the user to insert, update and delete films and its category as needed.

## Instructions

## Backend

- `GET		/api/films`
- `GET		/api/films/:id`
- `GET		/api/categories`
- `GET		/api/categories/:id`
- `POST		/api/films`
- `POST		/api/categories`
- `PUT		/api/films`
- `DELETE	/api/films`
- `GET		/api/films-categories`

All resources return JSON.

## Frontend

- `index.html`
- `films.html`
- `show-film.html`
- `insert-film.html`
- `update-film.html`
- `delete-film.html`
- `categories.html`
- `show-category.html`
- `insert-category.html`

Frontend must use Bootstrap
